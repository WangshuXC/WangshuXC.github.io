基础语法

Python的基础语法相较于其他语言更为简单，基本上可以在一个小时内掌握

字面量

在代码中被写下来的固定的值被称为字面量

Python中有以下6种数据类型：

数字（Number）

  • 整型（int）如：10、-10
  • 浮点型（float）如：3.14、-3.14
  • 复数（complex）如：3+4j
  • 布尔（bool）1为True，0为False

字符串（String）

字符串定义

字符串拼接

如若是讲两个字符串拼接，可用加号

name = "梁"+"晓储"

但若是字符串和非字符串变量拼接则需要使用字符串格式化

字符串格式化

若是有多个数据需要“填充”:

格式符号 转化
%s 将内容转换成字符串，放入占位位置
%d 将内容转换成整数，放入占位位置
%f 将内容转换成浮点型，放入占位位置

用%f进行字符串格式化时可能会用到数字精度控制

我们可以使用辅助符号”m.n”来控制数据的宽度和精度

m，控制宽度，要求是数字（很少使用）,若设置的宽度小于数字自身，不生效

.n，控制小数点精度，要求是数字，会进行小数的四舍五入

这种方法不理会类型、不做精度控制，适合对精度没有要求的时候快速使用

标识符

标识符命名规则：

  1. 只允许出现中英文、数字、下划线（_）

  2. 数字不可以用于开头

  3. 不可以使用关键字

运算符

运算符 描述
+ 加
- 减
* 乘
/ 除
// 取整除
% 取余
** 指数

常用语句

  1. type(x)

    用于查看变量的数据类型

  2. int(x)|float(x)|str(x)

    用于强制转换数据类型

  3. print(x)

    用于将x输出

  4. input()

    用于输入信息，可在括号内用字符串提示输入

    如：input("请输入你的名字：")
